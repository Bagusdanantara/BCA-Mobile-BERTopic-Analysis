# Analysis of BCA Mobile Reviews on Google Play Store
This repository contains the dataset and resources for the research titled: **"Identifying Problem Trends in Mobile Banking: A Temporal BERTopic Analysis of BCA Mobile Reviews"**.

## About the Dataset
* **Source:** Scraped from Google Play Store Reviews.
* **Target App:** BCA Mobile.
* **Period:** July 2025 - December 2025 (6 Months).
* **Data Type:** Raw text (Bahasa Indonesia).

## ⚠️ Privacy & Ethics Statement
To comply with ethical standards and privacy regulations (including GDPR and UU PDP Indonesia), this dataset has been **anonymized**.
* **User Identifiers Removed:** All `usernames`, `profile images`, and unique `IDs` have been permanently removed to protect user privacy.
* **Content Integrity:** The review content (`content` column) remains **raw and unedited** (preserving original typos, slang, and emojis) to ensure the authenticity of the NLP analysis.
